# Reinforcement Learning Projects

This repository contains practical implementations of reinforcement learning algorithms developed during my Master's in Artificial Intelligence (AI4CI) at CNAM Paris.

## Overview

The project focuses on training agents to learn optimal behavior through interaction with environments using reward-based learning.

## Implemented Algorithms

- Deep Q-Network (DQN)
- Proximal Policy Optimization (PPO)
- Advantage Actor-Critic (A2C) 

## Key Concepts

- State, action, reward framework
- Policy learning and optimization
- Exploration vs exploitation
- Reward shaping
- Environment interaction using Gymnasium

## Tools & Technologies

- Python
- Gymnasium / OpenAI Gym
- Stable-Baselines3
- NumPy
- Matplotlib

## Experiments

- Training agents in simulated environments
- Comparing performance of different RL algorithms
- Analyzing convergence and reward trends

## Results

- Agents successfully learned optimal strategies in given environments
- PPO showed stable learning behavior compared to DQN *(you can adjust this based on your results)*

## Applications

- Robotics navigation
- Autonomous decision-making systems
- Control systems optimization
